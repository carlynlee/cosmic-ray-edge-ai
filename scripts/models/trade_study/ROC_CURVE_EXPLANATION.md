# ROC Curve Trade Study Explanation

The ROC (Receiver Operating Characteristic) curves shown here compare the performance of 10 different binary classifier configurations for predicting cosmic ray coincidence events. Each curve plots the true positive rate (sensitivity) against the false positive rate (1 - specificity) across different classification thresholds, with the area under the curve (AUC) quantifying overall discriminative ability—higher AUC values indicate better performance, with 1.0 representing perfect classification and 0.5 representing random guessing. We evaluated variations across four key hyperparameters: **architecture size** (ranging from small [32, 16] to large [128, 64] neurons, including a deeper 3-layer variant [64, 64, 32]), **dropout regularization** (0.0, 0.3 baseline, and 0.5), **learning rate** (0.0001, 0.001 baseline, and 0.01), and **batch size** (16, 32 baseline, and 64). The results show that all configurations achieve similar performance with ROC-AUC values clustered around 0.81-0.82, indicating that the model is robust to hyperparameter variations and that the baseline configuration [64, 32] with dropout 0.3, learning rate 0.001, and batch size 32 provides a good balance between model complexity and performance for this binary classification task.
