# Architecture Comparison

We explored six different classifier architectures to identify the most effective approach for cosmic ray coincidence detection: a Multi-Layer Perceptron (MLP) neural network with two hidden layers (64 and 32 neurons), Logistic Regression with balanced class weights, Random Forest with 100 trees and depth-limited to 10, Gradient Boosting with 100 estimators, XGBoost, and LightGBM. All architectures achieved remarkably similar performance with ROC-AUC values clustered between 0.80 and 0.82, with the MLP neural network achieving the highest score of 0.8158. This convergence suggests that the task is well-suited to multiple learning paradigms, though the MLP's slight advantage, combined with its compact size (~4K-5K parameters) and suitability for edge deployment, makes it the optimal choice for onboard AI inference. The consistent performance across architectures also indicates that the five sensor features (ADC value, SiPM voltage, temperature, pressure, and acceleration) provide sufficient discriminative power for this binary classification task, regardless of the specific learning algorithm employed.
