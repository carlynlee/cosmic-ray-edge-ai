The Problem

Every minute, thousands of cosmic rays pass through each square meter of Earth. Each cosmic ray offers a clue about extreme astrophysical events like supernovae and active galactic nuclei. Today we can try to detect them with a patchwork of citizen-science smartphones and low-cost detectors, but the data is noisy, globally distributed, and overwhelmingly made of artifacts and background.

The question: How can we design a network of detectors and machine learning models so the system as a whole surfaces scientific insight?

Why This Project

  • Real infrastructure. Test your own hypotheses and ML models on a live pipeline running on the National Research Platform, with a path to deploy your work on SCinet (the Supercomputing conference's research network) in the future.
  • Real collaborators. Connects CPP with Caltech High Energy Physics and the global citizen-science network.
  • Real skills. Hardware + cloud + ML — the combination that's hard to get in any single class.
  • Real output. Contributions show up in papers, open-source repos, and conference demos you can point employers to.
